Yuki
====
Analyze all the videos in a folder,
export the thumbnails and output the Excel file.

![icon](https://github.com/loonghao/Yuki/blob/master/resources/yuki.png)

Features:
- Analyze videos.
- Export thumbnails.
- Save to Excel.

Request:
- addict-2.1
- python-2.7
- pyside-1.2
- yaml-1.2
- xlsxwriter-1.2
