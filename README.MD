Yuki
====
Analyze all the videos in a folder,
export the thumbnails and output the Excal file.

![Main GUI](https://github.com/loonghao/Yuki/blob/master/resources/yuki.png)

Features:
- Analyze videos.
- Export thumbnails.
- Save to Excal.
